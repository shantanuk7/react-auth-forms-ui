# Signup and Login Forms using React
This is a simple React based web application frontend that showcases singup and login forms.


## Project setup

### Requirements:
This project was created using `npm create vite@latest`, [Vite](https://vite.dev/) React library. with:
- node version `25.5.0`
- npm version `11.8.0`

After cloning, in the directory perform:

1. Install the npm packages
```bash
npm install
````

2. Run the app
```bash
npm run dev
```
